1

초기 로그인 방법

ℹ
Slack 초대 메일이 사전에 발송됩니다. 메일이 오지 않았다면 #poc-문의사항 채널 또는 IT 담당자에게 편하게 문의해 주세요.
1

초대 메일 확인

회사 이메일로 받으신 Slack 초대 메일을 열고 [지금 시작하기] 버튼을 클릭해 주세요.

⚠
초대 링크는 발송 후 72시간이 지나면 만료돼요. 받으셨다면 빨리 진행해 주세요!
2

비밀번호 설정

처음 사용할 비밀번호를 설정합니다. 아래 조건을 모두 충족해야 해요.

  • 영문 대·소문자, 숫자, 특수문자를 조합해서 만들어 주세요
  • 최소 10자리 이상으로 설정해야 합니다
  • 생년월일·이름·사번 등 개인정보는 포함하지 마세요
3

워크스페이스 접속

로그인하면 광동제약 Slack 워크스페이스가 자동으로 열립니다.

  • 웹 브라우저에서 바로 사용할 수 있어요
  • Slack 데스크톱/모바일 앱을 설치하면 더욱 편리하게 사용할 수 있어요
4

프로필 설정

오른쪽 상단 프로필 사진 → [프로필 편집]에서 아래 항목을 꼭 입력해 주세요.

  • 표시 이름: 실명 (예: 홍길동)
  • 소속: 부문 / 본부 / 부서 순으로 입력 (예: CMO부문 마케팅본부 브랜드팀)
  • 사번: 본인 사번 입력
2

Google Authenticator OTP 등록 방법

🔐
2단계 인증(2FA)은 필수예요! PoC에 참여하는 모든 분들은 처음 로그인 후 반드시 OTP를 등록해 주세요. 등록하지 않으면 계정 접근이 제한될 수 있습니다.
1

Google Authenticator 앱 설치

스마트폰에 앱이 없다면 먼저 설치해 주세요.

  • iPhone: App Store에서 "Google Authenticator" 검색 후 설치
  • Android: Google Play에서 "Google Authenticator" 검색 후 설치
2

Slack 2단계 인증 설정 페이지 이동

PC에서 아래 순서대로 따라해 주세요.

  • 오른쪽 상단 프로필 사진 클릭
  • [기본 설정] 선택
  • 왼쪽 메뉴에서 [보안] 탭 클릭
  • [2단계 인증 설정] 버튼 클릭
3

인증 앱 선택

인증 방법을 고르는 화면에서 "인증 앱 사용"을 선택하고 [계속]을 눌러 주세요.

4

QR 코드 스캔

Slack 화면에 QR 코드가 나타나면 아래 순서대로 진행하세요.

  • Google Authenticator 앱 실행
  • 오른쪽 하단 [+] 버튼 탭
  • "QR 코드 스캔" 선택
  • Slack 화면의 QR 코드를 카메라로 스캔
⚠
QR 코드는 절대 다른 사람에게 보여주지 마세요. 이 코드로 내 계정에 접근할 수 있습니다.
5

6자리 인증 코드 입력 및 완료

앱에 표시된 6자리 숫자 코드를 Slack 화면에 입력하고 [확인]을 눌러 주세요.

  • 코드는 30초마다 새로 바뀌어요. 시간 안에 입력해 주세요.
  • 등록이 완료되면 "2단계 인증이 활성화되었습니다" 메시지가 나타납니다.
✔
이후 Slack에 로그인할 때마다 앱에서 OTP 코드를 확인하여 입력하면 돼요.
6

백업 코드 저장

Slack에서 백업 코드를 제공해요. 스마트폰을 분실했을 때 계정을 복구할 수 있어요.

  • 코드를 인쇄하거나 안전한 곳에 메모해 두세요.
  • 절대 다른 사람에게 공유하지 마세요.
3

보안 정책 안내

광동제약 Slack PoC에서 적용되는 3가지 보안 정책을 안내드려요. 사용 중 보안에 대해 우려되는 부분이 있다면 참고해 주세요.

🔒

앱 설치 — 담당자 승인 방식

  • Slack과 연동되는 앱(Bot, 외부 서비스 등)은 담당자 확인 후 설치할 수 있어요
  • 사용하고 싶은 앱이 있다면 #poc-문의사항 채널로 편하게 문의해 주세요
  • 아직 승인되지 않은 앱은 자동으로 차단될 수 있습니다
📢

채널 유형 — 내부/외부 채널 구분

  • 내부 채널은 임직원만 참여할 수 있는 기본 채널이에요
  • 외부 채널은 외부 협력사와 소통이 필요할 때 개설할 수 있어요. 별도 승인 없이 직접 생성 가능합니다
  • 소속 본부·부문·팀 채널과 공통 채널(#전체공지, #poc-문의사항)은 처음부터 자동으로 보여요
🔗

Google Drive 링크 — 외부 공유 차단

  • PoC 기간 중에는 Google Drive 파일을 외부에 공유할 수 없어요
  • 사내 구성원끼리만 파일을 공유할 수 있습니다
  • 개인 Google 계정의 Drive 파일은 공유하지 않도록 해 주세요
  • 꼭 외부 공유가 필요한 경우에는 #poc-문의사항으로 문의해 주세요
4

채널 사용 가이드

ℹ
업무에 필요한 비공개 채널을 직접 생성하여 필요한 인원을 초대한 후 사용할 수 있습니다.

📌 공통 채널 (전체 참여자)

채널명유형용도대상
#전체공지 읽기전용 전사 공지사항 (관리자만 작성) 전체
#poc-문의사항 내부 PoC 관련 질문, 불편사항, 요청사항 전체

👥 내 팀 채널 (CMO 부문 예시)

채널명 패턴유형용도예시
#00_cmo_* 내부 부문/본부 전체 공유 채널 #00_cmo_전체
#01_cmo_* 내부 팀 전용 업무 채널 #01_cmo_마케팅

⚙️ 운영부서 채널 (AI센터·IT·정보보호)

채널명 패턴유형용도예시
#00_operate_전체 내부 운영부서 공통 공유 채널 #00_operate_전체
#01_operate_* 내부 팀별 운영 전용 채널 #01_operate_it

🔍 채널 유형 구분 기준

유형대상외부 초대개설 조건
내부 채널 광동제약 임직원 불가 운영부서 승인 후 개설
외부 채널 외부 협력사 포함 가능 채널 관리자가 채널 생성 후 필요한 인원만 초대
비공개 채널 초대된 멤버만 불가 채널 관리자가 채널 생성 후 필요한 인원만 초대
✔
올바른 채널 사용 체크리스트
  • 부문·본부 채널(#00_*)은 소속된 모든 구성원에게 내용이 공유돼요. 메시지 작성 전에 한 번 더 확인해 주세요
  • 외부 협력사와 소통이 필요하다면 외부 채널을 직접 만들고, 꼭 필요한 인원만 초대해 주세요
  • 외부 채널은 가끔씩 멤버를 확인하여 더 이상 필요 없는 인원은 내보내 주세요
  • 개인적인 업무 대화는 DM(다이렉트 메시지)을 활용해 주세요
  • Google Drive 파일을 공유할 때는 사내 구성원에게만 공유해 주세요. PoC 기간 중 외부 공유는 차단됩니다
