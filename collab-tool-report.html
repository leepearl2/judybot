광동제약 AI센터 · 기준 환율 1,450원/USD · 2026년 3월
🎯 왜 지금인가 — 디지털 워크플레이스 전략

협업툴 도입은 단순한 메신저 교체가 아닙니다. AI가 실제로 일할 수 있는 업무 인프라를 구축하는 것입니다. 현재 광동제약의 소통 방식은 메일·메신저·구두 보고로 파편화되어 있어 AI가 작동할 기반이 없습니다.

📧 커뮤니케이션 분절

메일, 메신저, 구두 보고 등 채널 다변화로 정보 히스토리 파편화. 결정 이력이 남지 않음.

📁 파일 버전 혼선

첨부파일 위주 공유로 '최종_최최종_진짜최종.pptx' 반복. 중복 작업 및 오류 위험.

🏭 현장-사무실 단절

품질관리 현장과 사무실 간 실시간 데이터 동기화 부족. 이슈 대응 지연.

AI센터가 그리는 그림

모든 소통·문서가
하나의 플랫폼에 축적
→
AI 에이전트가
그 위에서 실제로 일함
→
사람은 판단과
창의에 집중

GWS + Slack은 이 그림의 토대 레이어입니다. 이것 없이는 AI 에이전트, Gemini 활용, 자동화 모두 반쪽짜리가 됩니다.

⚡ 와우 시나리오 — 이렇게 달라집니다
SCENARIO 01

공장 품질 이상, 3분 내 전달

GMP 라인 온도·습도 이상 감지 즉시 품질관리 담당자 + 생산팀장 Slack 동시 알림. 현장 스마트폰으로 확인 → 조치 지시 → 완료 보고까지 하나의 채널에서. 이슈 대응 이력이 자동 기록되어 감사(Audit) 자료로 활용 가능.

대응 시간 단축 · GMP Audit 대응
SCENARIO 02

신제품 출시, 부서 사일로가 사라진다

출시 D-30. 마케팅·영업·생산·허가팀이 하나의 채널에서 진행상황 공유. 생산 일정 변경 → 영업팀 즉시 인지 → 마케팅 대응 → 허가 서류 공동 편집. "이거 누가 알고 있어요?" 질문이 사라짐.

크로스팀 지연 제거 · 출시 리드타임 단축
SCENARIO 03

임원이 매일 아침 브리핑을 받는다

매일 오전 8시, 임원 전용 채널에 자동 메시지 발송: 전국 영업 실적 요약 / 목표 대비 달성률 / 주요 이슈. ERP 연동으로 데이터 자동 집계. 별도 보고 대기 없이 현황 파악 후 하루 시작.

보고 대기 제거 · 의사결정 속도 향상
SCENARIO 04

온보딩이 자동으로 돌아간다

신입사원 입사 당일, 워크플로우 자동 실행. IT팀 계정 세팅 요청 / HR팀 교육 일정 안내 / 팀장 온보딩 체크리스트 발송이 동시에. 첫날부터 누락 없이 온보딩 완료.
