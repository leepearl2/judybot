공유된 NotebookLM 노트북 확인

AI센터에서 공유한 사내규정 노트북에 접속해서,
내 계정에 노트북이 정상적으로 공유되었는지 먼저 확인해 주세요.

🔗 공유 노트북 열기 (NotebookLM)
✓
링크를 클릭하면 회사 계정 (사번@kwangdong.mzpoc.xyz)으로 NotebookLM이 열리는지 확인
✓
노트북 내용(사내규정 소스 파일들)이 정상적으로 보이는지 확인
✓
접속이 안 되거나 내용이 보이지 않으면 AI센터 (#poc-문의사항)로 문의
💡 왜 확인하나요?
Gem에서 사내규정을 정확하게 안내하려면, NotebookLM에 공유된 규정집 노트북이 내 계정에서 보여야 합니다. 다음 단계에서 Gem의 지식 소스로 이 노트북을 연결하게 됩니다.
1

Gemini 접속 및 회사 계정 로그인 확인

gemini.google.com 접속 후 우측 상단 프로필에서 회사 계정으로 로그인되어 있는지 확인해 주세요.
로그인 확인 후 왼쪽 사이드바 Gems 옆 화살표(▶)를 클릭하고, + 새 Gem 버튼을 클릭해 주세요.

▲ 우측 상단 계정 확인 후 → 사이드바 Gems 클릭 → + 새 Gem 버튼 클릭

2

Gem 기본 정보 입력 후 지식에 NotebookLM 선택

Gem 편집 화면에서 아래 순서대로 진행해 주세요.

1
제목 입력란에 사내규정 안내 입력
2
설명 입력란에 광동제약 사내규정 안내 도우미 입력
3
요청 사항은 입력하지 않아도 됩니다 — NotebookLM에 미리 설정되어 있어요 😊
4
하단 지식 섹션의 + 버튼 클릭 → NotebookLM 선택

▲ 지식 > + 클릭 → NotebookLM 선택 (빨간 테두리 항목)

📌 경로: 지식 섹션 + 클릭 → 파일 업로드 / Drive에서 파일 추가 / 코드 가져오기 / NotebookLM 중 → NotebookLM 선택
3

사내규정 노트북 선택 후 추가

노트북 추가 팝업이 열리면 최근 목록에서
광동제약 사내규정 안내를 선택(✔)하고 추가 버튼을 클릭해 주세요.

▲ 「광동제약 사내규정 안내 · 소스 76개」 선택 → 추가 버튼 클릭

✓
목록에 사내규정 안내가 보이는지 확인
✓
노트북 선택 후 우측에 파란 체크(✔) 표시 확인
✓
하단 1개 선택됨 확인 후 추가 버튼 클릭
💡 목록에 노트북이 안 보인다면?
STEP 0으로 돌아가서 공유 링크로 먼저 접속해 주세요. 한 번 접속하면 목록에 나타납니다.
4

Gem 저장 (업데이트)

노트북이 지식 소스로 추가되면 우측 상단 업데이트 버튼을 클릭해 저장해 주세요.
이제 Gem에서 사내규정을 바로 물어볼 수 있습니다! 🎉
