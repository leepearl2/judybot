🟢 미래: AI가 함께 일하는 환경

  • 실시간 연결 — 클라우드 기반 즉시 소통·공유

→ Slack: 채널 기반 실시간 소통 + 스레드로 이력 자동 보존

→ Drive: 클라우드 단일 저장소 + 전문 검색 + 퇴사자 데이터 보존

🤖 AI 활용 기반 부재

현재 인프라에서는 AI 도구를 업무에 즉시 연동할 수 없음

문제: AI 활용이 개인 수준에 머무르고, 업무 프로세스에 내장 불가

→ GWS+Slack: Gemini·AI 에이전트가 업무 흐름 안에서 직접 동작

Slack 병행 테스트의 목적

Slack은 단순 메신저가 아닌, AI·시스템이 업무 흐름 안에서 동작하는 통합 인터페이스입니다

🤖 AI 에이전트 통합 허브

  • Slack은 글로벌 기업에서 AI 비서의 대화 창구로 활용 중
  • Salesforce Agentforce, OpenAI ChatGPT 등 AI 에이전트가 Slack 안에서 직접 동작
  • 직원이 채널에서 질문하면 AI가 사내 데이터를 검색·요약·답변
  • 광동 활용: 논문 요약, 시장 데이터 조회, 보고서 초안 생성을 Slack 채널 안에서 수행

🔗 시스템 연동 플랫폼

  • ERP·MES·SFA 등 기존 시스템의 알림과 워크플로우를 한 곳으로 통합
  • 공장 장애 → 자동 알림 → 스레드 대응 → 화상회의(Huddle) 즉시 전환
  • 외부 파트너와 Slack Connect로 보안 유지하며 협업
  • 광동 활용: 현장 이슈 실시간 대응, 대행사 협업 채널 운영

📌 Salesforce · Agentforce

Slack 채널에서 AI 에이전트가 CRM 데이터 조회·고객 응대 자동화

📌 OpenAI · ChatGPT for Slack

채널 대화 요약, 문서 초안 생성, 리서치를 Slack 안에서 즉시 수행

📌 Anthropic · Claude for Slack

장문 분석, 코드 리뷰, 보고서 검토를 팀 채널에서 바로 요청
