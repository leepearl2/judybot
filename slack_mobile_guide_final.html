모바일 로그인 가이드

URL 직접 입력 방식 · ekdp.enterprise

1
플레이스토어에서 슬랙 설치
아직 앱이 없다면 먼저 설치하세요

📱 Google Play 스토어를 열고 검색창에 slack을 입력하세요.

⬇️ Slack – SLACK TECHNOLOGIES 앱을 찾아 [설치] 버튼을 누르세요.

✅ 이미 설치되어 있다면 이 단계는 건너뛰고 바로 앱을 실행하세요!
↓
2
앱 실행 후 [계속하기] 누르기
슬랙 시작 화면

🚀 슬랙 앱을 실행하면 시작 화면이 나타납니다.

하단의 [계속하기…] 버튼을 누르세요.

↓
3
[워크스페이스 URL 입력] 클릭
로그인 화면 하단 링크

🔗 이메일 입력 화면 하단의 [문제가 있나요? 워크스페이스 URL 입력] 링크를 누르세요.

💡 이메일을 입력하지 않아도 됩니다. 바로 하단 링크를 누르세요.
↓
4
워크스페이스 주소 입력
회사 URL 직접 입력

🔗 입력창에 아래 주소를 입력하세요.

(입력창 오른쪽에 .slack.com이 이미 표시되어 있어요)

📋 ekdp.enterprise 를 입력하세요

입력 후 [계속] 버튼을 누르세요.

↓
5
광동제약 로그인 방식 선택
SSO 또는 이메일/비밀번호

🏢 광동제약에 로그인 화면이 나타나면 원하는 방식을 선택하세요.

• [광동제약을(를) 사용하여 로그인] → 회사 SSO 인증

• [이메일과 비밀번호로 로그인하기] → 이메일/비밀번호 방식

↓
6
이메일 입력 후 로그인
SSO 선택 시 진행되는 화면

📧 SSO 로그인을 선택하면 이메일 입력 화면으로 이동합니다.

회사 이메일을 입력하고 [로그인] 버튼을 누르세요.

예: leepearl@ekdp.com

↓
7
인증 앱에서 6자리 코드 입력
2단계 인증 완료

🔐 인증 앱으로 이동 화면이 나오면, Microsoft Authenticator 또는 Google Authenticator에서 6자리 코드를 확인하세요.

코드를 입력하고 [계속] 버튼을 누르면 로그인 완료! 🎉

✅ 이제 슬랙을 사용할 수 있습니다!

문의사항은 IT팀 또는 슬랙 관리자에게 연락하세요.
워크스페이스: ekdp.enterprise.slack.com
