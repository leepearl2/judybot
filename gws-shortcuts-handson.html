🏋️ Sheets 실습해보기

  1. A1에 "이름", B1에 "날짜", C1에 "금액" 입력 후 Tab으로 이동해보기
  2. B2에서 Ctrl+;로 오늘 날짜 자동 입력하기
  3. C2:C5 범위에 숫자 입력 후 Ctrl+Shift+5로 퍼센트 서식 적용
  4. 행 선택 후 Ctrl+Alt+Shift+=으로 행 삽입해보기

🏋️ Slides 실습해보기

  1. Ctrl+M으로 슬라이드 3장 빠르게 추가하기
  2. 2번 슬라이드 선택 후 Ctrl+D로 복제해보기
  3. 텍스트 박스 여러 개 선택 후 Ctrl+Alt+G로 그룹화해보기
  4. Ctrl+Shift+F5로 발표 모드 시작 후 B 눌러보기

⚡ 생산성 꿀팁

🔍 @멘션으로 팀원 태그하기
문서나 댓글에서 @이름을 입력하면 팀원을 태그할 수 있어요. 태그된 사람에게 알림이 가요!
📎 다른 파일 연결하기
Docs에서 @를 입력하면 Drive 내 다른 파일, 이벤트, 회의 등을 바로 연결할 수 있어요.
🔗 링크 공유 시 보기/댓글/편집 권한 선택
공유 → 링크 설정에서 권한 수준을 꼭 확인하세요. 외부 공유 시 "뷰어"로 설정하는 습관을!
