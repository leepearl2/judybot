1
Chrome 프로필 생성 및 로그인
회사 계정 전용 Chrome 프로필을 새로 만들어 로그인합니다
Chrome 우상단 프로필 아이콘 클릭 → [Chrome 프로필 추가]

Chrome 브라우저 우측 상단의 프로필 아이콘을 클릭하면 계정 목록이 나타납니다. 맨 아래 [Chrome 프로필 추가]를 클릭합니다.

💡 TIP  기존 개인 계정과 회사 계정을 분리하기 위해 새 프로필을 만드는 것을 권장합니다.
새 Chrome 프로필 설정 창 → [로그인] 클릭

새 Chrome 프로필 설정 창이 열립니다. [로그인] 버튼을 클릭해 회사 계정으로 연결합니다.

💡 TIP  [로그아웃 상태 유지]를 선택하면 계정 연동 없이 프로필만 생성됩니다. 반드시 [로그인]을 클릭하세요.
회사 이메일 주소 입력 → [다음]

Chrome에 로그인 화면에서 회사 이메일 주소를 입력하고 [다음]을 클릭합니다.

⚠ 주의  반드시 회사 계정(사번@kwangdong.mzpoc.xyz) 형식으로 입력하세요. 개인 Gmail 계정이 아닙니다.
기존 비밀번호 입력 → [다음]

이메일 입력 후 비밀번호 입력 화면이 나타납니다. 현재 회사 계정 비밀번호를 입력하고 [다음]을 클릭합니다.

💡 TIP  비밀번호를 잊었다면 AI센터에 문의해 주세요.
새 비밀번호 설정 → [다음]

처음 로그인하는 경우 안전한 새 비밀번호 설정 화면이 나타날 수 있습니다. 8자 이상의 새 비밀번호를 입력하고 확인란에 동일하게 입력한 후 [다음]을 클릭합니다.

⚠ 주의  영문+숫자+특수문자를 조합한 8자 이상의 비밀번호를 설정하세요. 설정한 비밀번호는 반드시 기억해 두세요.
조직 관리 안내 확인 → [계속]

조직(kwangdong.mzpoc.xyz)에서 프로필 정보와 기기 정보를 관리할 수 있다는 안내 화면이 나타납니다. 내용 확인 후 [계속]을 클릭합니다.

💡 TIP  이 화면은 회사 조직 계정으로 관리되는 정상적인 안내입니다. [계속]을 클릭하면 됩니다.
로그인 완료 확인 → [Google 계정 관리] 클릭

로그인이 완료되면 우측 상단에 프로필 이름이 표시됩니다. 팝업에서 [Google 계정 관리]를 클릭해 계정 설정 페이지로 이동합니다.

✅ 완료  로그인 완료! 이제 OTP 2단계 인증 등록을 진행합니다.
2
OTP 키 등록
Google 계정 보안 설정에서 OTP 앱으로 2단계 인증을 등록합니다
Google 계정 홈 → 왼쪽 메뉴 [보안 및 로그인] 클릭

Google 계정 관리 페이지(myaccount.google.com)에서 왼쪽 메뉴의 [보안 및 로그인]을 클릭합니다.

💡 TIP  주소창에 myaccount.google.com 을 직접 입력해도 됩니다.
보안 및 로그인 → [2단계 인증] 클릭

Google에 로그인하는 방법 섹션에서 [2단계 인증] 항목을 클릭합니다. 현재 '2단계 인증이 사용 중지되었습니다'로 표시됩니다.

⚠ 주의  현재 상태가 '사용 중지'로 표시됩니다. 이 항목을 클릭해야 등록이 시작됩니다.
2단계 인증 설정 화면 → [OTP 앱 추가] 클릭

2단계 인증 설정 화면에서 아래 '2단계' 섹션의 OTP → [OTP 앱 추가]를 클릭합니다.

💡 TIP  패스키/보안 키, Google 메시지 등 여러 방법이 있지만 OTP 앱이 가장 안전하고 안정적입니다.
OTP 앱 화면 → [+ 인증자 설정] 클릭

OTP 앱 페이지가 열립니다. Google Play 또는 App Store에서 Google OTP 앱을 먼저 설치한 후, [+ 인증자 설정]을 클릭합니다.

⚠ 주의  앱이 설치되지 않은 경우 먼저 설치하세요.
Android: Google Play → Google OTP 검색 설치
iPhone: App Store → Google Authenticator 검색 설치
QR 코드 스캔 후 앱의 6자리 코드 입력 → [인증]

화면에 QR 코드가 표시되면 Google OTP 앱에서 + → QR 코드 스캔으로 스캔합니다. 앱에 표시된 6자리 코드를 입력하고 [인증]을 클릭합니다.

⚠ 주의  코드는 30초마다 자동 갱신됩니다. QR 코드는 타인에게 절대 공유하지 마세요.
[사용 설정] 클릭 → OTP 등록 완료

인증 완료 후 OTP 앱 화면에서 [사용 설정] 버튼을 클릭합니다. 'OTP — 방금에 추가됨'이 표시되면 등록이 완료된 것입니다.

✅ 완료  OTP 등록 완료! 이제 다음 로그인부터는 비밀번호 입력 후 OTP 앱의 6자리 코드를 입력해야 합니다.

💡 TIP

자주 묻는 질문
Q. Chrome 프로필을 새로 만드는 이유가 뭔가요?
개인 Google 계정과 회사 계정을 브라우저에서 완전히 분리하기 위해서입니다. 프로필별로 북마크, 비밀번호, 설정이 독립적으로 관리됩니다.
Q. Google OTP 앱은 어디서 받나요?
Android: Google Play 스토어에서 "Google OTP" 검색 → 설치
iPhone: App Store에서 "Google Authenticator" 검색 → 설치
Q. OTP 코드 6자리를 입력했는데 오류가 납니다
코드는 30초마다 바뀝니다. 새 코드가 생성될 때까지 기다렸다가 다시 입력해 보세요. 스마트폰 시간이 정확한지도 확인하세요.
Q. 스마트폰을 교체하면 OTP가 사라지나요?
새 스마트폰에 Google OTP 앱 설치 후, Google 계정 보안 설정에서 [OTP 앱 추가]를 다시 진행하면 됩니다.
Q. 도움이 필요하면 어디에 문의하나요?
AI센터로 문의해 주세요. 현장 지원도 가능합니다.
