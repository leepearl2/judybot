🖥️ 2. 최초 PC 허용 작업 (로그인 시 최초 1회)
⚠️
로그인 후 "액세스가 차단되었습니다" 화면이 뜨는 경우 아래 순서대로 따라하시면 됩니다. 이 작업은 처음 한 번만 하면 돼요.

🔧 처리 절차

  1. Chrome 웹 스토어에 접속합니다.
  2. Endpoint Verification 확장 프로그램을 검색 후 설치합니다.
  3. 설치 완료 후 확장 프로그램에서 Sync now 버튼을 클릭합니다.
  4. Gmail, 드라이브 등 GWS 서비스 접속이 허용되는지 확인합니다.
🚫
그래도 차단이 계속되는 경우 위 절차 수행 후에도 접속이 차단된다면 AI센터로 문의해 주십시오.
↑ 목차로
🔐 3. 최초 OTP 등록 작업 (최초 1회)
⚠️
GWS 로그인 시 OTP 인증이 필요해요 처음 로그인할 때 OTP 등록을 먼저 완료해야 이후 정상적으로 사용할 수 있어요. 스마트폰에 Google OTP 앱을 미리 설치해 두세요!

🔧 OTP 등록 절차

  1. GWS 로그인: PC에서 GWS 사이트에 접속하여 ID/PW를 입력해 로그인합니다.
  2. OTP 등록 팝업: OTP 인증 페이지가 나오면 "OTP 등록하기" 버튼을 클릭합니다.
  3. QR 코드 스캔: 화면에 나타나는 QR 코드를 스마트폰의 Google OTP 앱으로 스캔합니다.
  4. 인증번호 입력: 앱에서 생성된 6자리 숫자를 GWS 인증 화면에 입력 후 등록을 완료합니다.
🚫
OTP 등록 후에는 로그인할 때마다 앱에서 숫자를 확인해야 해요 스마트폰을 분실하거나 앱을 삭제한 경우 AI센터로 문의해 주세요.

📁 5. GWS 활용 정책 및 보안 수칙

📂 2.1 파일 저장 원칙

  • 업무 파일은 반드시 구글 드라이브(내 드라이브 또는 공유 드라이브)에 저장하세요
  • 개인 클라우드(네이버 클라우드, 드롭박스 등)나 USB에 저장·복사하는 것은 금지입니다
  • 팀이 함께 쓰는 파일은 공유 드라이브에 저장하는 것을 권장합니다

🌐 2.2 외부 공유 통제 원칙

  • GWS 파일을 회사 외부(타 회사 이메일 등)로 공유하는 것은 기본적으로 막혀 있습니다

🔑 2.3 로그인 보안 원칙

  • 로그인 시 OTP(2단계 인증) 사용이 필수입니다 — 비밀번호 외에 인증 앱의 숫자 코드를 추가로 입력하는 방식이에요
  • OTP 설정을 하지 않으면 로그인이 차단됩니다
  • 내 업무에 필요한 파일·폴더에만 접근하고, 다른 부서 파일을 무단으로 열람하지 마세요

🤖 2.4 AI 서비스 사용 원칙

  • 구글 AI(Gemini, NotebookLM)에 1등급(비밀)·2등급(부서비) 문서 내용을 입력하는 것은 금지입니다
  • 어떤 등급인지 모르겠다면 아래 보안 등급 표를 참고하세요

📋 2.6 활동 기록 보관 원칙

  • 드라이브·Gmail·Meet·Gemini 등 GWS 사용 기록은 최소 1년간 보관됩니다
  • 보안 정책을 위반한 이력도 별도로 기록됩니다

💻 2.7 사용 기기 원칙

  • 미리 등록·승인된 기기에서만 GWS에 접속할 수 있습니다
  • 회사에서 지급한 PC는 자동으로 등록됩니다
  • 개인 PC나 휴대폰으로 접속하려면 AI센터에 별도로 승인 요청이 필요합니다

🏷️ 6. 문서 보안 등급 분류 기준
🚫
AI 서비스(Gemini, NotebookLM) 입력 주의 1등급(비밀)2등급(부서비) 에 해당하는 내용은 AI 서비스에 입력하지 마십시오.
등급 해당 문서 유형 AI 서비스 입력
1등급 (비밀) 주요 재무·회계 정보 / 합병·매각 관련 / 중요 계약·프로젝트 / 의약품·연구개발 정보 / 기타 비밀 처리 필요 문서 금지
2등급 (부서비) 임직원 인사정보 / 영업 정책 / 고객 정보 / 시스템 운영·설계 / 계약서·협약서·각서 / 대외 공표 금지 업무 계획 / 기타 부서 중요 문서 금지
3등급 (사내비) 사내 유통·공유가 허용되는 일반 업무 문서 허용
